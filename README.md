# Whisper Rooms 

A real-time anonymous room-based chat application currently in development.

This project is being built from scratch as a learning journey into full-stack and mobile app development.

##Concept

Whisper Rooms is designed to allow users to:

- Create private chat rooms
- Join rooms using a unique code
- Enter a nickname (no login required)
- Get approval from the room creator before joining
- Chat in real-time
- Have rooms automatically deleted when inactive

The goal is to build a lightweight, privacy-focused communication platform.

## Planned Tech Stack

Frontend:
- Flutter (Dart)
- Android-first development

Backend:
- Node.js
- WebSocket 


## Planned Features

- [ ] Room creation with auto-generated code
